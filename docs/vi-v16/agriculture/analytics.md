# Phân tích

Phần này của module Nông nghiệp được sử dụng để theo dõi dữ liệu và tạo báo cáo.

Tất cả dữ liệu liên quan đến các DocType sau đây có thể được theo dõi bởi ERPNext:

* Phân tích cây trồng
* Phân tích đất
* Phân tích nước
* Kết cấu đất
* Thời tiết
* Tiêu chí phân tích nông nghiệp
