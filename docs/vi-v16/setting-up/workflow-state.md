# Trạng thái Quy trình (Workflow State)

**Trạng thái Quy trình là một trạng thái cụ thể trong một quy trình được thiết lập.**

Các Trạng thái Quy trình khác nhau có thể được đạt được trước hoặc sau khi áp dụng các Hành động Quy trình khác nhau. Nếu bạn muốn tạo một Quy trình có nhiều cấp phê duyệt (ví dụ: từ quản lý, quản lý cấp cao, tổng quản lý, v.v.), bạn có thể thiết lập các trạng thái tương ứng trong Trạng thái Quy trình.

Theo mặc định, các Trạng thái Quy trình sau đây được tạo sẵn trong ERPNext:

![Workflow States](https://raw.githubusercontent.com/frappe/erpnext_documentation/master/erpnext_documentation/www/docs/v13/assets/img/setup/workflow-states.png)

Các Trạng thái Quy trình có thể có các màu sắc khác nhau tùy theo trạng thái:

* Success - Xanh lá
* Danger - Đỏ
* Inverse - Đen
* Primary - Xanh dương đậm
* Info - Xanh dương nhạt
* Warning - Cam

Trạng thái tài liệu:

* Lưu (Saved) = 0
* Xác nhận (Submitted) = 1
* Hủy (Canceled) = 3
