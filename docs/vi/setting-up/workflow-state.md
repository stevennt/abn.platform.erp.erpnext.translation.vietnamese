# Trạng thái Quy trình (Workflow State)

**Trạng thái Quy trình là một trạng thái trong quy trình mà bạn tạo ra.**

Các Trạng thái Quy trình khác nhau có thể đạt được trước hoặc sau khi áp dụng các Hành động Quy trình khác nhau lên chúng. Nếu bạn muốn tạo một Quy trình có nhiều cấp phê duyệt từ quản lý, quản lý cấp cao, tổng quản lý, v.v., bạn có thể thiết lập các trạng thái cho nó từ Trạng thái Quy trình.

Theo mặc định, các Trạng thái Quy trình sau đây được tạo sẵn trong ERPNext:

![Workflow States](https://docs.erpnext.com/docs/v13/assets/img/setup/workflow-states.png)

Các Trạng thái Quy trình có thể có các màu sắc khác nhau tùy theo trạng thái.

* Success - Xanh lá
* Danger - Đỏ
* Inverse - Đen
* Primary - Xanh dương đậm
* Info - Xanh dương nhạt
* Warning - Cam

Trạng thái tài liệu:

* Saved = 0
* Submitted = 1
* Canceled = 3
