#Tích hợp ERPNext với các ứng dụng khác

Hiện tại, ERPNext đã có sẵn các tích hợp mặc định cho một số ứng dụng như Shopify, cổng SMS và cổng thanh toán của bạn. Để tích hợp ERPNext với các ứng dụng khác, bạn có thể sử dụng REST API của Frappe. Kiểm tra các liên kết sau để tìm hiểu thêm về REST API của Frappe.

[Frappe Rest API](https://frappe.io/docs/v13/user/en/guides/integration/rest_api.html)
