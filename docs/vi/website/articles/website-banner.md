# Thay đổi kích thước Banner Website

Mỗi website của tài khoản ERPNext đều được tự động tạo ra từ đó. Trên một website, logo được thiết lập dựa trên hình ảnh logo đã chọn trong Trình hướng dẫn thiết lập (Setup Wizard). Bạn có thể thay đổi hoặc chỉnh sửa thuộc tính cho logo công ty của mình từ phần Cài đặt Website (Website Settings).

`Khám phá > Website > Cài đặt Website`

Để biết các bước chính xác về cách tải lên Banner Website và thay đổi kích thước của nó, vui lòng tham khảo hướng dẫn dưới đây.
