# Đổi tên hàng loạt

Sử dụng công cụ Đổi tên hàng loạt, bạn có thể chọn để chỉnh sửa/thay đổi nhiều ID tài liệu cùng một lúc. Công cụ này chỉ dành cho Người dùng đã được chỉ định vai trò Quản trị viên hệ thống (System Manager).

### Công cụ Đổi tên

Bạn có thể đổi tên ID của tối đa 500 bản ghi cùng một lúc. Dưới đây là các bước để đổi tên hàng loạt các bản ghi. Giả sử chúng ta đang đổi tên Mã mặt hàng (Item Code) cho các mặt hàng hiện có.

#### Bước 1: Mở tệp Excel

Trong một tệp bảng tính, hãy nhập các ID Mặt hàng cũ vào cột đầu tiên và các ID Mặt hàng mới vào cột thứ hai. Lưu tệp bảng tính dưới định dạng CSV.

![Upload Data](https://docs.erpnext.com/docs/v13/assets/img/using-erpnext/using-bulk-rename-1.png)

#### Bước 2: Tải lên tệp dữ liệu

Để tải lên tệp dữ liệu, hãy đi tới:

> Cài đặt > Dữ liệu > Đổi tên hàng loạt (Bulk Rename)

Chọn DocType mà bạn muốn đổi tên. Ở đây, DocType sẽ là Mặt hàng (Item). Sau đó, Duyệt và Tải lên tệp dữ liệu.
