# Quản lý khoản vay

Phân hệ Quản lý khoản vay trong ERPNext giúp bạn quản lý các khoản vay của mình ngay từ Đơn đăng ký vay cho đến khi Tất toán khoản vay. Bạn có thể theo dõi việc giải ngân, hoàn trả, thế chấp tài sản đảm bảo, giải chấp, dồn tích lãi vay và nhiều nội dung khác.
