# Client Script

**Client Script là các đoạn mã nhỏ được thực thi trên trình duyệt để mở rộng hoặc tùy chỉnh các chức năng tiêu chuẩn của ERPNext.** 

> Trang chủ > Tùy chỉnh > Custom Script

![Client Script](https://raw.githubusercontent.com/frappe/erpnext_documentation/master/erpnext_documentation/wwwhttps://raw.githubusercontent.com/frappe/erpnext_documentation/master/erpnext_documentation/www/docs/v13/assets/img/customize/customize-erpnext-client-scripts.png)

## 1. Cách tạo Custom Script

Tạo một Custom Script (bạn phải có vai trò Quản trị viên hệ thống - System Manager để thực hiện việc này):

  1. Đi tới Danh sách Custom Script và nhấn vào Mới.
  2. Bạn sẽ được chuyển hướng đến một trang yêu cầu nhập DocType mà bạn muốn tạo Custom Script.
  3. Nhập Custom Script của bạn và đảm bảo rằng nó đã được bật.
  4. Lưu.

## 2. Ghi chú

  * Server Custom Script chỉ dành cho Quản trị viên (Administrator).
  * Client Custom Script sử dụng JavaScript và Server Custom Script sử dụng Python.
  * Để thử nghiệm, hãy đảm bảo đi tới Công cụ > Xóa bộ nhớ đệm (Clear Cache) và làm mới trang sau khi cập nhật một Custom Script.
